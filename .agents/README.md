# Reborn Wave Ruflo Agent Team

This folder contains a Ruflo-style marketing and investor acquisition team for Reborn Wave Group.

Use it after setting up Ruflo in the project:

```powershell
npx ruflo@latest init wizard
```

Recommended swarm shape:

- `reborn-growth-queen`: owns weekly strategy, priorities, and approvals.
- `reborn-content-producer`: creates multilingual short-form content and captions.
- `reborn-social-amplifier`: plans likes, shares, community prompts, and posting rhythm.
- `reborn-investor-leads`: builds investor-facing outreach and follow-up flows.
- `reborn-partnerships`: finds local Batam partners, hosts, singers, creators, salons, and event collaborators.
- `reborn-analytics-optimizer`: tracks campaign numbers and recommends next actions.

Keep compliance human-reviewed. Investment content must say ROI is a target, not guaranteed return, and every public claim should be checked before posting.
